Neural network vs transformer

A transformer is a neural network — but a special kind. Here is the plain version next to the transformer our SLM uses, with the one part that makes the difference highlighted.

Plain neural net

Layers of neurons, fully connected by learned weights. Great for a fixed input (an image, a row of numbers). It has no idea about order and the inputs never look at each other.

Self-attention

The new ingredient. Every token looks at the other tokens and pulls in what is relevant, so meaning depends on context. Positional embeddings add the sense of order.

So what is a transformer?

A neural network plus self-attention. The feed-forward block inside each layer is an ordinary neural net — attention is simply wrapped around it, repeated a few times.
