How this model was trained

SLM 4096 is a real (tiny) GPT, built to be fully understandable. It does one thing — predict the next character — and it learned it from exactly one page of memory. Here is the entire recipe, no hand-waving.

The recipe

  1. Tokenize by character. Every unique character in the corpus becomes a number — 54 in total. No fancy sub-word tokenizer.
  2. Predict the next character. We feed the model 128 characters and ask it to guess character 129, over and over.
  3. Measure surprise. Cross-entropy loss scores how wrong each guess was. A clueless model scores ln(54) ≈ 3.99.
  4. Nudge the weights. AdamW adjusts all 211,712 parameters to reduce that loss. Repeat 3,000 times — a few seconds on a laptop CPU.
  5. Memorize, on purpose. With only 4 KB of data the model overfits and learns the page by heart. In a real LLM that's a bug; here it's the whole joke.

Where 211,712 comes from

d = 64  (embedding width)

embeddings   : vocab·d + ctx·d
             : 54·64 + 128·64        =  11,648
4 blocks     : 4 × (12·d² + 13·d)    = 199,936
final norm   : 2·d                   =     128
head         : tied to embeddings    =       0
-----------------------------------------------
TOTAL                                = 211,712

Rule of thumb: params ≈ layers × 12 × d². Widen d or stack more layers and you walk from 211K to billions.

The loss curve (real run)

iter     1 | loss 3.9995  ← random (ln 54)
iter   500 | loss 1.1311
iter  1000 | loss 0.3639
iter  1500 | loss 0.2585
iter  2000 | loss 0.1897
iter  3000 | loss 0.1450  ← page memorized

Loss falling from ~3.99 to ~0.15 is the model learning its 4 KB by heart. That's why it answers everything in the same satirical voice.

Bonus: neural network vs transformer

A transformer is a neural network — with one addition: self-attention. Here is the plain version next to the transformer our SLM uses, with the one part that makes the difference highlighted.

Plain neural net

Layers of neurons, fully connected by learned weights. Great for a fixed input (an image, a row of numbers). It has no idea about order and the inputs never look at each other.

Self-attention

The new ingredient. Every token looks at the other tokens and pulls in what is relevant, so meaning depends on context. Positional embeddings add the sense of order.

So what is a transformer?

A neural network plus self-attention. The feed-forward block inside each layer is an ordinary neural net — attention is simply wrapped around it, repeated a few times.
